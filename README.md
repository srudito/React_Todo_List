# Todo List App (React)

A simple and clean Todo List application built with React that allows users to create headings and manage multiple list items under each heading.

---

## Features

- Add multiple todo categories (headings)
- Add multiple list items under each heading
- Delete entire headings
- Clean and responsive card layout
- Beginner-friendly React structure

---

## Built With

- React (Functional Components + Hooks)
- CSS (Flexbox)

---

## How It Works

- Each heading acts as a card
- Each card contains:
  - A list of items
  - Input field to add new items
  - Button to delete the entire heading
- State is managed using useState
